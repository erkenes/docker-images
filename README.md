---

This image was built for the [Neos-Flow-Framework](https://flow.neos.io/) and the [Neos-CMS](https://www.neos.io/) but can used for any other project as well.

It supports the following addons:
- composer
- git
- pickle
- imagick, gd
- redis
- pdo
- opcache
- soap
- curl
- openssl
- yaml
- open-ldap
